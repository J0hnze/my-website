# Brand Color Palette

- **Primary Accent:** #00e6c3 (Cyan/Teal)
- **Background:** #181a1b (Dark)
- **Card/Section BG:** #23243a
- **Text:** #f5f5f5 (Main), #b0b8c1 (Secondary)
- **Tag BG:** #222327
- **Tag Text:** #00e6c3

Use only these colors for backgrounds, text, and accents. Avoid introducing new colors except for code syntax highlighting.
