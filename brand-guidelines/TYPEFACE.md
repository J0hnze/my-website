# Brand Typeface Guidelines

- **Primary Font:** Inter, Roboto, Segoe UI, Arial, sans-serif
- **Monospace (code/tags):** Fira Mono, Consolas, Menlo, monospace

**Rules:**
- Use only the primary font for all headings, body, and navigation.
- Use the monospace font only for code blocks, inline code, and tag elements.
- Do not introduce additional fonts or font weights beyond regular, medium, and bold.
