# Frontend Design Skill

This skill helps with:
- Designing modern, accessible, and responsive web frontends
- Generating HTML, CSS, and JavaScript for UI components
- Prototyping layouts and wireframes
- Reviewing and critiquing frontend code for best practices
- Suggesting improvements for accessibility and usability

## Example Usage
- "Create a responsive navigation bar for a blog."
- "Suggest improvements for this CSS grid layout."
- "Generate a color palette for a tech website."
- "Review this HTML for accessibility issues."

## Guidelines
- Follow modern web standards (HTML5, CSS3, ARIA)
- Prioritize accessibility (WCAG 2.1)
- Use semantic HTML and scalable CSS
- Prefer mobile-first, responsive design
- Explain design choices when possible
- Suggest improvements for clarity, usability, and performance
